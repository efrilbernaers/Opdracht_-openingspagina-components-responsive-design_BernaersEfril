First, install:

```bash
npm i
```

Next, run the development server:

```bash
npm run dev
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Toelichting 

Voor deze opdracht heb ik de drie aangeleverde artikels uitgewekrt als webpagina’s. Ik heb vooral gefocust op een duidelijke structuur, goede leesbaarheid en eenvoudige navigatie. 

Op de homepage heb ik een korte introductie geplaatst. Zodat gebruikers meteen weten waar de website over gaat. Daaronder staan de artikels in een overzichtelijke lijst met titel en afbeelding. Door gebruik te maken van klikbare kaarten (met hover-effect) wordt het duidelijk dat je kan doorklikken. Dit maakt de navigatie intuïtief en gebruiksvriendelijk. In de artikels zelf heb ik gekozen voor een vaste en herkenbare opbouw: rubriek, titel, intro, auteur, afbeelding en daarna de tekst. Zo weten gebruikers altijd waar ze bepaalde informatie kunnen vinden. De belangrijkste elementen staan bovenaan. Zodat de inhoud meteen duidelijk is.

De tekst is opgesplitst in korte paragrafen met voldoende witruimte (space-y en leading-relaxed). Dit zorgt ervoor dat de artikels makkelijker leesbaar en minder zwaar zijn. Bij langere artikels heb ik ook tussenkopjes gebruikt om de inhoud beter te structureren.

Voor de visuele hiërarchie heb ik verschillende tekstgroottes gebruikt. De titel is groot en opvallend. De intro iets groter dan de gewone tekst en tussenkopjes zorgen voor extra duidelijkheid. Hierdoor kan de gebruiker snel door de tekst scannen.

Ik heb afbeeldingen toegevoegd om de inhoud visueel te ondersteunen en afwisseling te brengen in lange teksten. De captions onder de afbeeldingen geven extra uitleg en context.

Daarnaast heb ik interactieve elementen toegevoegd, zoals de knoppen “Luisteren” en “Delen”. Deze zorgen voor extra functionaliteit en maken de pagina dynamischer. Door hover-effecten krijgen gebruikers ook visuele feedback bij interactie.

Onderaan elk artikel staan “Lees ook”-links. Zodat gebruikers makkelijk kunnen doorklikken naar andere artikels. Dit helpt bij de navigatie en zorgt ervoor dat gebruikers langer op de website blijven.

Voor de layout heb ik gewerkt met een maximale breedte (max-w-3xl / max-w-4xl) en een gecentreerde opbouw. Dit zorgt ervoor dat de tekst comfortabel leesbaar blijft op grotere schermen. Met flexbox heb ik elementen zoals artikels en afbeeldingen naast elkaar geplaatst waar nodig.

Ik heb gebruik gemaakt van herbruikbare structuren en componenten. Zoals gelijkaardige layouts voor elk artikel en een consistente stijl voor knoppen en kaarten. Hierdoor blijft de website uniform en overzichtelijk.

Tot slot heb ik Tailwind CSS gebruikt voor de styling. Dit zorgt voor consistente typografie, kleuren en spacing. De layout past zich ook aan verschillende schermgroottes aan. Waardoor de website goed bruikbaar blijft op desktop, tablet en mobile.
